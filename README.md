# Overview

The main website for the `merc` project, built using [MkDocs](https://www.mkdocs.org/) and hosted on GitHub Pages. It uses the  [Material](https://squidfunk.github.io/mkdocs-material/) theme for a modern design.

## Commands

The documentation can be served locally or built using the following commands:
* `mkdocs serve` - Start the live-reloading docs server.
* `mkdocs build` - Build the documentation site.
